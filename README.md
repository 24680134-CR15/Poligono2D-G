# Polígono 2D en Blender con Python

Este proyecto muestra cómo crear un polígono regular en 2D dentro de Blender utilizando un script en Python.

## Contenido
- `Poligon2D`: Script que genera un polígono regular (hexágono por defecto).
- `Poligono2D.blend`: Archivo de Blender con el resultado del script.

## Uso
1. Abrir Blender.
2. Ir a la pestaña **Scripting**.
3. Abrir el archivo `Poligon2D`.
4. Ejecutar el script con **Run Script**.

## Herramientas
- Blender
- Python (bpy)
